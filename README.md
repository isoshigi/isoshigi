### FlowMatic

https://github.com/KaungSetLinn/FlowMatic

学校のチーム開発(二年次)で作成したプロジェクト管理グループウェア

バックエンドチームの管理を担当。
メンバーの学習コストを低減してプロジェクト成功率を向上するため、学校の授業で取り扱ったPythonと強力なテンプレートがあるDjangoを採用。
また、UX設計にも関与し、「学生」という曖昧なペルソナ設定から、「学校の授業でチーム制作に取り組む経験の浅い学生グループで、使えるツールや考え方がばらばらで、限られた時間と授業コマ数内で最大限の成果を挙げなければならない」という内容にすることでUXの方向性をまとめ、対象にとって使いやすいアプリケーションにすることができた。全学年対象の学内発表会に出場。

Tools: Python, Django Rest Framework, SQLite
